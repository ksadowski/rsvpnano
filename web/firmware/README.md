# Web Flasher Artifacts

This directory contains the browser-flasher image plus the app-only OTA binary:

- `rsvp-nano.bin`
- `rsvp-nano-ota.bin`

It is generated by:

```sh
python3 tools/export_web_firmware.py
```

`rsvp-nano.bin` is the full merged image for fresh installs from the browser flasher.
`rsvp-nano-ota.bin` is the app-only binary for GitHub Release OTA updates.

The generated `.bin` files are intentionally ignored by Git. GitHub Pages builds and publishes the web flasher artifacts as part of the site deploy.
